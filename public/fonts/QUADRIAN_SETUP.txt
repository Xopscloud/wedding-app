QUADRIAN FONT SETUP
===================

To add the Quadrian font to your project:

Option 1: Download from Google Fonts (Recommended)
1. Visit: https://fonts.google.com/
2. Search for "Quadrian"
3. Click on the font and select "Download family"
4. Extract the ZIP file
5. Locate the Quadrian Regular (400) font file in WOFF2 format
6. Copy it to this folder (public/fonts/)
7. Rename it to "Quadrian.woff2"

Option 2: Use a local font file
1. If you have a Quadrian.woff2 file, place it in this folder (public/fonts/)
2. Restart your Next.js dev server
3. The @font-face declaration in styles/globals.css will automatically load it

Option 3: Use DaFont or other free font sources
1. Search "Quadrian" on DaFont.com or similar sites
2. Download the font in WOFF2 format if available
3. Place it in public/fonts/ and name it "Quadrian.woff2"

After placing the font file:
- Reload your browser (hard refresh: Ctrl+Shift+R or Cmd+Shift+R)
- The landing page names should now display in Quadrian font
- If not visible, check browser DevTools Console for font loading errors

Current Setup:
- Font location: public/fonts/Quadrian.woff2
- Font fallback: Allura (from Google Fonts)
- CSS Rule: @font-face defined in styles/globals.css
